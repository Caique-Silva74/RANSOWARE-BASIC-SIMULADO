Este projeto contém scripts desenvolvidos exclusivamente para fins educacionais e de demonstração de segurança. O objetivo é simular, em um ambiente rigorosamente controlado, como a criptografia de arquivos é implementada e, mais importante, como pode ser revertida usando uma chave de segurança.

encrypt_simulator.py
Demonstração de Ataque (Simulado): Script que realiza a criptografia de um conjunto pré-determinado de arquivos de teste.

decrypt_utility.py
Ferramenta de Recuperação (Defesa): Script que localiza a chave (.key) e a utiliza para reverter o processo de criptografia, demonstrando a recuperação de dados.
